## CMS Tuition Class Service
This is tuition class microservice of the Class Management System

## Requirements
* Open JDK 11
* Spring boot 2.7.9
* MySQL 8.0
* Apache Maven 3.9.0



## Build

```
mvn clean compile package
```

## Run

```
mvn spring-boot:run
```

or

```
java -jar target/cms-tuitonclass-service-0.0.1-SNAPSHOT.jar
```

## Test

```
mvn test
```
